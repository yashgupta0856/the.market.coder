# QuantFusion — VCP Market Intelligence Platform

QuantFusion is a research-grade, end-to-end quantitative equity analysis system focused on identifying Volatility Contraction Pattern (VCP) stocks and ranking them using sector intelligence and sector rotation.

This project is designed as a production-style institutional research system.

---

## Core Philosophy
- Deterministic, explainable logic
- No look-ahead bias
- Sector-aware ranking
- Capital-weighted intelligence
- Web-based dashboard

---

## Architecture Flow
Data → Indicators → VCP Scanner → Sector Intelligence → Sector Rotation → Stock Ranking → Web UI

---

## Phases
Phase 1: Data ingestion  
Phase 2: Indicators  
Phase 3: VCP scanner  
Phase 4: Sector intelligence + rotation  
Phase 5: Stock scoring & ranking  

---

## Sector Rotation
Uses:
- Breadth
- Capital-weighted impact
- Relative strength vs benchmark

Produces deterministic rotation scores.

---

## Web Application
Built using FastAPI + Jinja2.
Read-only dashboard.
CSV files generated by backend pipelines only.
